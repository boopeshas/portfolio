# Boopesh A S — Personal Portfolio

![Portfolio Preview](https://via.placeholder.com/800x400/0F172A/00CFFF?text=Boopesh+Portfolio)

Modern, highly animated personal portfolio website built with pure **HTML, CSS, and Vanilla JavaScript**.

## ✨ Features

- **Premium Dark Theme** with cyan accents
- Smooth typing animation for roles
- Glassmorphism cards + floating particles
- Interactive timeline & project cards
- Fully responsive (Mobile → Desktop)
- Scroll animations & progress bar
- No frameworks — lightweight & fast

## 🛠️ Tech Stack
- HTML5
- CSS3 (Custom + Glassmorphism)
- Vanilla JavaScript
